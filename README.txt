Il vous suffit de créer une base de données ayant le nom : bdd.

Le dossier contient un fichier sql : BDD.sql.
Celui-ci contient le code sql à insérer dans la base de donnée.

Une fois celà exécuté, il vous suffit d'inclure l'ensemble des fichiers et dossier du projet dans wamp, xampp ou autre.
Le projet est fonctionnel.
